# How to Create Your GitHub Profile README

This file contains the GitHub profile README I created for you, ready to be deployed to your profile!

## Quick Setup Instructions

1. **Create a new repository on GitHub:**
   - Go to https://github.com/new
   - Repository name: `joebertj` (must match your GitHub username exactly)
   - Description: `README` or `My GitHub Profile`
   - Choose **Public**
   - **DO NOT** initialize with README, .gitignore, or license
   - Click "Create repository"

2. **Clone and setup locally:**
   ```bash
   git clone git@github.com:joebertj/joebertj.git
   cd joebertj
   ```

3. **Copy the README:**
   ```bash
   cp README_PROFILE.md README.md
   ```

4. **Commit and push:**
   ```bash
   git add README.md
   git commit -m "Add GitHub profile README"
   git push -u origin main
   ```

5. **Verify:**
   - Visit https://github.com/joebertj
   - Your profile README should now appear at the top!
